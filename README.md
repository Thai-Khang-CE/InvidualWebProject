# Seasonal Wardrobe

## Project Information
- Project name: Seasonal Wardrobe
- Assigned domain: D2 - Fashion / Shoe Store
- Concept: A fashion store organized by four seasonal collections: Spring, Summer, Autumn, Winter
- Tech stack: HTML, CSS, JavaScript, PHP, MySQL
- Local environment: XAMPP, Apache, MySQL, phpMyAdmin

## Local Running Instruction
1. Place the project folder inside `D:\xampp\htdocs\`
2. Start Apache in XAMPP
3. Open `http://localhost/seasonal-wardrobe/public/` in the browser
